# 开发参考 Demo

本目录存放 RAGFlow 集成的开发参考代码，**不在生产环境中运行**。

生产环境的 RAGFlow 工具函数位于：`tools/ragflow_tools.py`

## 文件说明

- `chat_assistant_demo.py` — 聊天助手查询与提问 Demo
- `knowledge_demo.py` — 知识库创建与文件上传 Demo

## 运行方式

```bash
# 在项目根目录下
python scripts/demos/chat_assistant_demo.py
python scripts/demos/knowledge_demo.py
```
